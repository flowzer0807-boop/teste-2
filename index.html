⚙️ Configurar banco de dados

Para que os dados fiquem sincronizados em todos os dispositivos, configure o Firebase (gratuito pelo Google). Faça isso uma única vez.

1
Acesse console.firebase.google.com → clique em Adicionar projeto → dê um nome (ex: grupo-icf)
2
No projeto criado, clique em Autenticação → Começar → ative o método E-mail/senha
3
Clique em Firestore Database → Criar banco de dados → escolha Modo de produção → selecione a região southamerica-east1
4
Em Firestore → aba Regras, cole e publique:
allow read, write: if request.auth != null;
5
Vá em Configurações do projeto (ícone ⚙️) → Seus apps → clique em </> Web → registre o app → copie o objeto firebaseConfig e cole abaixo
💡 Grátis para sempre no plano Spark do Firebase — suporta até 1 GB de dados e 50.000 leituras/dia, mais que suficiente para o Diário de Obras.
